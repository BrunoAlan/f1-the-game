# F1 The Game

A browser-based Formula 1 race management and strategy game with a pixel-retro aesthetic.

## What is this?

Take control of a real 2026 F1 team and manage your race weekend: gather data in Practice, push for pole in Qualifying, plan your pit strategy, and make real-time decisions during the Race. Features all 11 teams and 22 drivers from the 2026 F1 season.

## Features (MVP)

- **Practice** — Auto-lapping minigame that reveals tire degradation data
- **Qualifying** — Choose your risk level: Safe, Push, or Full Send
- **Strategy Room** — Plan pit stops with stint sliders and compound selectors
- **Race Simulation** — Watch positions change in real-time, control Push/Neutral/Save modes, call pit stops
- **Dynamic Weather** — Rain can arrive mid-race, forcing tire strategy changes
- **Safety Car** — Incidents compress the field, creating strategic pit windows
- **22-car grid** — All 2026 F1 teams and drivers with performance-inspired stats

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Zustand
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
pnpm install
```
